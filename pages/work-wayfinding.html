WayFinding

Library Navigation System

A centralized kiosk synced with a mobile app that improves physical library wayfinding through interactive maps and navigation.

Problem

Students often struggle to navigate the library, leading to frustration, wasted time, and underutilization of library resources. The library lacks an accessible wayfinding system integrated with its existing database.

Goal

Create a solution that improves physical navigation in the library while increasing efficiency and accessibility.

Stakeholders

  • Students
  • Library visitors
  • Librarians

Methodology

  • User interviews
  • Observation
  • Contextual research

Key insights

  • Library anxiety deters people from using available resources.
  • Users feel uncomfortable repeatedly asking librarians for help.
  • The library book naming convention is confusing for many users.
  • Most visitors go directly to the front desk for navigation assistance.
  • Participants often photograph book numbers on their phones while navigating.

Features

  • Interactive kiosk navigation
  • QR code synced mobile navigation
  • Indoor positioning system
  • Accessibility-focused haptic feedback
  • Book search and guided routing

Key screens

Kiosk Search

Search for books and resources

Interactive Map

Displays route and navigation

Mobile Navigation

Guided indoor directions through QR sync

Project planning

Users search for books at a centralized kiosk. The kiosk generates a map and QR code that syncs directions to the user’s phone.

Technology stack

  • Bluetooth beacon network for positioning
  • Mobile motion sensors (gyroscope, accelerometer, magnetometer)
  • Hybrid indoor positioning system
  • Proximity haptic feedback for accessibility
  • Middleware for routing and beacon management

Metrics

Reduced Navigation Time

~9.5 minutes

70% faster navigation

Ease of Navigation Improvement

64%
